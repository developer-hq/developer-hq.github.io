The beta version of typora is expired, please download and install a newer version

免费版不能用了，找到解决方法，记录一下

按Windows+R打开运行窗口，输入regedit，点确定，

打开注册表，依次展开计算机\HKEY_CURRENT_USER\Software\Typora，

然后在Typora上右键，点权限，选中Administrtors，把权限全部设置为拒绝。
